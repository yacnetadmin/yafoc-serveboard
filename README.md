# YAC ServeBoard
Hybrid volunteer sign-up tool for York Alliance Church.

## Directories
- /frontend  → GitHub Pages web app (staff + volunteer portals)
- /api       → Azure Functions for data and authentication
- /docs      → Architecture and setup notes
